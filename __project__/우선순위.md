# WeLive 도메인 설계 우선순위

1. 공통 기반 (가장 먼저)
2. 계정/인증 도메인 (`User`, `AuthSession`, `Auth`)
3. 아파트/게시판 기초 도메인 (`Apartment`, `Board`)
4. 공지사항 도메인 (`Notice`)
5. 민원 도메인 (`Complaint`)
6. 투표 도메인 (`Poll`)
7. 댓글 도메인 (`Comment`)
8. 알림 도메인 (`Notification`)
9. 아파트 일정 도메인 (`ApartmentSchedule`)
